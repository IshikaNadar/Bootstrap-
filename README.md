# Bootstrap 5 Revision Demos

A collection of practice demos created while revisiting **Bootstrap 5 fundamentals**.  
This repo highlights responsive layouts and core UI components used in modern web development.

---

## 📌 Overview
This repository contains small, focused examples showcasing:
- **Grid System** – responsive layouts and alignment
- **Navbar** – collapsible navigation menus
- **Forms** – styled inputs, selects, checkboxes, and validation
- **Buttons & Alerts** – interactive UI elements
- **Cards & Badges** – content containers with status indicators

Each demo is designed to be **clean, minimal, and practical** for recruiters, peers, and collaborators.

---

## 🚀 Features
- Responsive design using Bootstrap 5
- Easy‑to‑understand code snippets
- Screenshots included for quick preview
- Beginner‑friendly structure for revision and sharing
